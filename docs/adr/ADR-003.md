# ADR-003: Observer Pattern con Spring Events

## Estado

Aceptado — 2026-05-17

## Contexto

El sistema necesita notificar multiples canales cuando un pedido es procesado.

## Decision

Se usa ApplicationEventPublisher y @EventListener para implementar Observer Pattern.

## Consecuencias

### Positivas

- Notificaciones desacopladas.
- Facil agregar nuevos listeners.
- Escalable.

### Negativas

- Debugging mas complejo.
- Orden de ejecucion no garantizado.

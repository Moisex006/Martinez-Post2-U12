# ADR-002: Strategy y Factory para procesar pedidos

## Estado

Aceptado — 2026-05-17

## Contexto

Existen distintos tipos de pedido con algoritmos de calculo diferentes.

## Decision

Se implementa Strategy Pattern combinado con Factory para seleccionar procesadores dinamicamente.

## Consecuencias

### Positivas

- Facil agregar nuevos tipos.
- Menor acoplamiento.
- Cumplimiento de Open/Closed.

### Negativas

- Mayor cantidad de clases.
- Configuracion mas compleja.
